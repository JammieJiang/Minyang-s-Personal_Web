# Minyang-s-Personal_Web
Nice to meet u!I'm Minyang,a student in SWUFE-UD
这是我的简洁个人简历网站，展示个人信息、技能和联系方式。

## 功能
- 📷 个人信息展示（照片+基本信息）
- 📥 简历PDF下载
- 📧 联系方式（邮箱、电话等）
- 📱 完全响应式设计

## 技术栈
- HTML5
- CSS3
- Vanilla JavaScript

## 快速开始
1. 克隆或下载本项目
2. 用浏览器打开 `index.html` 即可

## 文件结构
```
personal_web/

├── styles/

├── assets/
```
